Letreiro

🐝

Forma palavras com 7 letras num favo de mel. Inclui sempre a letra central!

📖 Como Jogar

  • 🔤 Forma palavras com mínimo 4 letras
  • ⭐ A letra central (dourada) é obrigatória
  • 🔁 Podes repetir letras na mesma palavra
  • 🎉 Pangrama = usar todas as 7 letras → bónus +7 pts!
  • 📊 4 letras = 1 pt · 5+ letras = 1 pt por letra

🎮 Modos de Jogo

🌸 Clássico — Fácil

Descobre todas as palavras sem limite de tempo. Letras reutilizáveis. Ideal para relaxar.

🔥 Clássico — Médio

Mais palavras para encontrar. Precisas de vocabulário sólido!

💎 Clássico — Difícil

Centenas de palavras possíveis. Para verdadeiros génios da língua!
